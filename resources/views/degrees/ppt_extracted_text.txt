Form Validation in Laravel
Validation ensures that user inputs meet certain rules before being processed.
Validation is usually handled in the controller.
Basic Validation Example
public function store(Request $request) {
    $request->validate([
        'name' => 'required|min:3',
        'email' => '
required|email
',
        'password' => 'required|min:6'
    ]);
    return "Form Submitted Successfully";
}

Eloquent Relationships in Laravel
What is Eloquent ORM?
Laravel uses 
Eloquent ORM (Object-Relational Mapping)
 to interact with databases using PHP objects instead of raw SQL.
Eloquent relationships allow you to define how different database tables are connected and interact with each other in an intuitive, object-oriented way.
Why Use Eloquent Relationships?
Eliminates complex SQL joins 
Improves code readability 
Makes database queries reusable 
Supports powerful features like eager loading and constraints
